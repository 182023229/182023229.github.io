Markdown语法
2022-01-15

工欲善其事必先利其器，写博客之前先学习点Markdown语法。

标题

Markdown标题有两种格式。

1.使用=和-标记

在标题的下一行使用=单独隔开表示为一级标题，-为二级标题。

2.使用#标记

使用#可表示1-6级标题，一级标题为一个#号，依次类推。

段落隔离

3.1 分割线

在一行中使用3个以上的*(星号)、-（减号）、_(下划线)来表示分割线，行内不能有其他东西。

3.2 删除线

如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线 ~~。

列表

Markdown支持有序列表和无序列表。
无序列表使用*（星号）、+（加号）、-（减号）作为标记，在标记后加个空格，再填写内容。

有序列表使用数字加上.号作为标记，也需在标记后加个空格。

代码

短块代码

可以使用`（反引号）将代码块包起来。

代码区块

使用四个空格或一个Tab键表示一个代码块

    public static void main(String args []){
    System.out.println("Hello World");
    }

使用```包裹一段代码，可指定一种语言：

System.out.println("Hello World");
